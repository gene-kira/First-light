⭐ WHAT Codex Purge Shell v4.6 IS
Codex Purge Shell v4.6 is a Windows 11 security appliance, implemented as a single monolithic Python file, combining:

A security dashboard

A system adjustment console

A threat scoring engine

A Suricata event viewer

A forensic snapshot system

A honeypot emulator

A GUI integrity watchdog

A swarm sync visualizer

A threat timeline (new in v4.6)

It’s essentially your own local SIEM + EDR + admin console, but safe (no destructive actions).

1. Gives you full Windows control from one GUI
You can adjust:

Services

Scheduled tasks

Defender real‑time protection

Network adapters

Power plans

Windows Update

Registry keys

Firewall rules

This replaces:

regedit

Resurrection behavior

Risk score

3. Creates forensic snapshots
It captures:

Firewall rules

Firewall profiles

Registry firewall policy

Metadata about why the snapshot happened

Delta metadata showing what changed

Rate‑limited (1 per hour)

Triggered only by persistent changes

Optional for Suricata/Windows Update

Compressed into ZIP

Restorable

This gives you rollback capability and forensic evidence.

It reads Suricata’s eve.json and shows:

Signature

Severity

MITRE tactic

Threat score

Suggested remediation

This gives you IDS visibility without needing a full SIEM.

5. Tracks honeypot behavior
Logical honeypot sessions track:

Attacker commands

Duration

Protocol tags

Suspicious behavior

Useful for analyzing attacker intent.

6. Protects GUI integrity
If any GUI element disappears or breaks:

It logs the issue

Requests a snapshot (cooldown‑controlled)

Attempts to restore the widget

Marks it red

This protects against:

GUI corruption

Tampering

Rendering issues

A logical “distributed node” view showing:

Local threat scores

Local snapshots

Local signatures

Persona state

Simulated peer nodes
